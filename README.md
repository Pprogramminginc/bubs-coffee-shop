# Bub's Coffee Shop ☕

A premium, cozy café landing page designed as a modern front-end portfolio project.

## 🌐 Overview
Bub's Coffee Shop is a polished, pet-friendly café concept built to feel like a real business website — not just a template. The design focuses on warmth, usability, and strong visual hierarchy.

## ✨ Features
- Modern responsive landing page
- Hero section with structured layout and branding
- Coffee menu + dog menu
- Gallery and about sections
- Interactive modal popups
- Smooth scroll + reveal animations
- Clean UI with card-based design

## 🛠 Built With
- React
- Vite
- CSS3
- JavaScript

## 🎯 Purpose
This project was created to demonstrate:
- Front-end design skills
- Layout and spacing control
- UI/UX thinking
- Real-world business presentation
